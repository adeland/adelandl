## Overview

**Crime Mapper Boston** is a machine learning system that forecasts crime patterns across Boston's police districts. Built for city resource allocation, the platform analyzes **204,356 crime records** across **14 districts** over **824 days** using Facebook's Prophet model to predict future crime trends and DBSCAN to locate geographic hotspots.

> **Note:** This project was developed as a collaboration between Simon Chen, Kelvin Kuang, Kelvin Yeung, and Daniel Ooi for a graduate data science course at Boston University.

---

## Key Features

- **District-Based Forecasting**: Generates individual Prophet models tailored to the unique seasonal and weekly crime patterns of each police district.
- **Spatial Hotspot Analysis**: Uses DBSCAN clustering to group crime incidents geographically (e.g., highlighting shoplifting hubs).
- **Automated Performance Safeguards**: Evaluates predictions against baseline averages and flags irregular/unpredictable districts to ensure decision safety.
- **Interactive Dashboard**: A Streamlit interface displaying time-series forecasts, historical trends, and dynamic geographic overlays.

---

## Technical Architecture

The system pulls data dynamically from the Boston.gov CKAN API and processes it through a two-stage analytical pipeline:

1. **Temporal Modeling (Prophet)**:
   - Configured with yearly and weekly seasonality.
   - Tailored to capture holiday effects (e.g., July 4th anomalies).
   - Generates confidence intervals for uncertainty quantification.

2. **Spatial Clustering (DBSCAN)**:
   - Groups geo-coordinates of specific crime types.
   - Filters noise to map high-density clusters.

## Performance Benchmarks

The Prophet models yielded a **14.3% average improvement** over naive baseline methods.

| Metric | System Average | Definition / Meaning |
| :--- | :--- | :--- |
| **MAE** | 3.38 crimes/day | Average prediction deviation per district |
| **RMSE** | 4.29 crimes/day | Standard deviation of prediction residuals |
| **MAPE** | 35.85% | Mean absolute percentage error |
| **Coverage** | 75.2% | Percentage of actual values within confidence bounds |

### Top Performing Districts
- **District C6**: **59.9% improvement** over baseline (MAE: 3.89)
- **District E13**: **59.8% improvement** over baseline (MAE: 3.19)
- **District A1**: **59.2% improvement** over baseline (MAE: 4.68)
